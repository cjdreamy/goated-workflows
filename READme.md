# lookout 
Stop shipping "hope-based" code. 🛡️

As a backend dev, I care more about System Integrity than button colors. Today, I built an automated "Gatekeeper" for my Node.js projects.

The Problem: Manual security audits are slow, and "spaghetti code" creeps in when you’re moving fast.
The Solution: A GitHub Action that runs on every PR to:

- 1. Run npm audit to block high-risk vulnerabilities.

- 2. Enforce strict ESLint rules (Quality Gate).

- 3. Automate the feedback loop by posting a report directly to the PR.

Logic is easy. Infrastructure is what makes a professional. 😤
